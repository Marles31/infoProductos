# Workflows

Los workflows de esta carpeta son SOPs ejecutables por un agente.

## Regla base (obligatoria)

Antes de ejecutar una tool, primero se lee la skill indicada en el workflow.

Formato esperado en cada workflow:

1. Objetivo
2. Entradas
3. Skills a leer (en orden)
4. Tools/comandos a ejecutar (en orden)
5. Validaciones de salida
6. Manejo de errores
