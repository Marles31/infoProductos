# Workflows

Aqui van los SOPs en Markdown.

Cada workflow debe definir:

1. Objetivo
2. Entradas requeridas
3. Herramientas (`tools/`) a ejecutar
4. Salidas esperadas
5. Manejo de errores y edge cases
