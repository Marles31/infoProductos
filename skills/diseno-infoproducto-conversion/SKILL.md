# Diseno Infoproducto Conversion

## Objetivo

Entregar piezas visuales con estructura comercial, no documentos tecnicos.

## Reglas de diseno

1. Definir direccion visual:
- 1 paleta principal (3 a 5 colores)
- 1 color acento de conversion
- escala tipografica clara (titulo, subtitulo, cuerpo, microcopy)

2. Diseñar por bloques:
- hero de impacto
- problema
- solucion
- accion inmediata
- prueba social/credibilidad
- CTA

3. Evitar contenido interno:
- no mostrar campos como `audiencia`, `objecion`, `signals`, etc.
- usar lenguaje de lector final

4. Mantener legibilidad:
- maximo 2 ideas por pantalla
- bullets cortos
- contraste alto

## Flujo recomendado

1. Generar copy final para lector.
2. Maquetar en HTML visual por secciones/paginas.
3. Inyectar imagenes reales o placeholders de estilo.
4. Revisar conversion: headline, beneficio, urgencia, CTA.

## Criterio de aprobacion

Si parece "guia interna", se rechaza.
Si parece "mini landing premium", se aprueba.
