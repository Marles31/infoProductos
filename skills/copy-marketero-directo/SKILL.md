# Copy Marketero Directo

## Objetivo

Convertir ideas en copy que persuade y mueve a accion.

## Estructura minima por pagina

1. Titular potente
2. Subtitulo con beneficio concreto
3. Desarrollo breve orientado a accion
4. Lista de puntos de valor
5. Micro-CTA o CTA principal

## Reglas de redaccion

- Hablarle al lector, no describir procesos internos.
- Escribir con verbos de accion.
- Usar resultados concretos (tiempo, claridad, seguridad, control).
- Evitar tecnicismos innecesarios.
- Cerrar cada bloque con siguiente paso claro.

## Anti-patrones

- "Esta pagina explica..."
- "Objetivo comercial..."
- "Audiencia: ..."
- repetir plantillas vacias sin aporte

## Criterio de calidad

Si el lector puede decir "esto me sirve hoy", el copy pasa.
